1) Для первичного запуска приложения, необходимо выполнить миграции для создания необходимых таблиц;
2) Выполнить скрипты из папки database\new_install;
3) Запустить парсер можно через консоль либо через строку https://rentapartaments.ga/upload_data/2. Последний параметр в строке запроса указывает на страницу,
с которой будет запускаться парсер. Если не указать параметр, то запуск начнется с 1 страницы.
Настройки количества страниц за 1 проход парсера указаны в файле app.php в параметре default_page_parse;

Демо версия
На демоверсии настроено два cron через планировщик хостинга. Первый - это проверка новых объявлений. Второй очищает признак новое объявление раз в сутки. 
Данный признак устанавливается на сутки.
